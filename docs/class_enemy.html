Позволяет получить дополнительные характеристики, которые должны добавиться к текущим перед ударом. Учитывает все эффекты, которые накладываются на врага перед ударом (настраиваются в константе Enemies::EFFECT_RELATION) c учётом их частоты вызывания.

Аргументы
targetцель, на которую действует персонаж
move_numberномер хода, для передачи в функции-эффекты
Возвращает
pawn_specifications - Характеристики персонажа, которые нужно добавить к текущим перед ударом
Example

Используется только в функции Pawn::attacked_by(Pawn&, int), также есть усложнённый аналог для игрока Player::get_extra_specifications()

Замещает Pawn.
