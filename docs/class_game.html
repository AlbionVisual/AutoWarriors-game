◆ play_round()

int Game::play_round ( bool is_last = false)
private

Запускает бой со случайным врагом, а затем проводит диалог по улучшеиям

Аргументы
is_lastПоследний ли бой (спрашивать ли смену орудия у пользователя)
Возвращает
0 - Бой закончился победой игрока
1 - Бой закончился поражением игрока
Example

Используется только в Game::start()

Проиграем 5 боёв:

for (int i = 0; i < 5; i++) {
code = play_round(game, player, enemy_list, enemies_amount, i == 4);
if (code == 1) break;
}
