Подробное описание

Ряд функций-постэффектов (как они выглядят показано ниже), тип-функция (тоже имеет ту же структуру), перечисление тегов (соответствующий каждой функции для хранения и быстрого доступа к ним при помощи хэш-таблиц), а также несколько вспомогательных функция для определения вида функции.

Автор
Vlad Korzun (vlad..nosp@m.korz.nosp@m.un.mi.nosp@m.nsk@.nosp@m.gmail.nosp@m..com)

Функция-эффект имеет вид:

pawn_specifications post_effect_func(const Pawn& who_activated, const Pawn& damager, int extra_damage, int move_number, printer_type external_printer)

Функция-постэффект будет использоваться для определения дополнительных характеристик персонажей во время получения урона от другого персонажа. Так её можно будет применить к любому персонажу, и описывать логику для каждого эффекта можно отдельно и независимо от персонажей
