◆ attacked_by()

void Pawn::attacked_by ( Pawn & attacker,
int move_number )

Основная функция персонажа для проведения боя. Тут вычисляется шанс попадания удара (при помощи полей specifications.agility у первого и второго персонажей), затем применяются эффекты по-очереди при помощи переписываемых методов Pawn::apply_damage(const Pawn&, int, int), Pawn::get_extra_specifications(Pawn&, int) и аналогичных у Player.

Аргументы
attackerТот кто атакуюет персонажа, доступ к которому будет получен через this
move_numberНомер хода для передачи в эффекты
Example

Из main.cpp:

pl2->attacked_by(*pl1, move_number);
